<h1>Ejecutar en Desarrollo</h1>

1. Clonar el repo

2. Ejecutar
    ``` npm i ```
3. Tener Nest Cli Instalado
    ``` npm i @nestjs/cli ```
4. Levantar DB
    ``` docker-compose up -d ```
